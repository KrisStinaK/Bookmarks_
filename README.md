# Bookmarks - Социальный сайт на Django

Bookmarks — это социальная платформа, где пользователи могут делиться изображениями, создавать закладки для понравившихся страниц и взаимодействовать с другими участниками. Проект реализован на Django и предназначен для удобного обмена визуальным контентом и организации ссылок.

---

## Основные возможности

- Регистрация и вход пользователей
- Загрузка и отображение изображений
- Создание и управление закладками (сохранение ссылок)
- Просмотр ленты с изображениями и закладками
- Лайки и комментарии к изображениям
- Личный профиль пользователя
- Поиск по изображениям и закладкам
- Административная панель для модерации контента

---

## Технологии

- **Backend:** Django, Django REST Framework (опционально)
- **Frontend:** HTML, CSS, Bootstrap
- **База данных:** SQLite

---

## Установка и запуск проекта

1. Клонируйте репозиторий:
```bash
git clone https://github.com/KrisStinaK/Bookmarks_.git
```

2. Перейдите в директорию проекта:
```bash
cd bookmarks
```

3. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv env
source env/bin/activate  # Linux / macOS
env\Scripts\activate     # Windows
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Выполните миграции базы данных:
```bash
python manage.py migrate
```
6. Создайте суперпользователя для доступа к административной панели:
```bash
python manage.py createsuperuser
```

7. Запустите сервер разработки:
```bash
python manage.py runserver
```
8. Перейдите по адресу: http://127.0.0.1:8000/ в браузере для использования сайта.
